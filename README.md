> An interactive Power BI dashboard analyzing sales, profit, and quantity from the Superstore dataset — segmented by Region, State, Category, and Order Date.

---

## 📌 What This Project Does

Transforms raw Superstore sales data into a stakeholder-ready interactive dashboard. Business users can filter by region, category, and segment to instantly see performance metrics and trends.

---

## ✨ Features

- 🔍 Dynamic slicers — filter by Region, Category, and Segment
- 📐 DAX measures for Total Sales, Profit, Quantity, and Profit Ratio
- 📈 Trend analysis via bar charts, line graphs, pie charts, and map visuals
- 🗺️ Geographic sales distribution on map visual
- 🎨 Optimized layout for clear business storytelling

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Power BI | Dashboard creation & visualization |
| DAX | Custom measures & calculations |
| Power Query | Data cleaning & transformation |

---

## 📊 Key Metrics Tracked

| Metric | Description |
|--------|-------------|
| Total Sales | Revenue across all regions and categories |
| Total Profit | Net profit with regional breakdown |
| Profit Ratio | Profit as % of sales |
| Quantity Sold | Units sold by category and segment |
